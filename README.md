
This is a project for recreation of the classic game Space Invaders arcade game built using python and pygame.
This project demonstrates core game development concepts such as sprite handling, collision detection, event loops, and player/enemy mechanics.

 Features
- Player Controls
  - Move left/right using arrow keys
  - Shoot bullets with the space bar (limited to 2 bullets, with cooldown)
- Player Stats
  - 3 lives
  - Respawn with temporary invincibility and flashing effect
  - Score tracking 
- Invaders
  - 3 rows of enemies with different sprites
  - Grid layout with configurable spacing
  - Top-row invaders randomly shoot bullets
- Bullets
  - Player bullets move upward
  - Enemy bullets move downward
  - Collision detection removes bullets and targets
- Scoreboard
  - Lives and score displayed in the top-left corner
- Game Over
  - Ends when player runs out of lives

 Requirements
- Python 3.8+
- PyGame library
